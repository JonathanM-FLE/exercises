A2

La forme négative avec "être"

Grammaire française - Niveau A2

📚 Rappel

Forme affirmative : Je suis content.

Forme négative : Je ne suis pas content.

Structure : sujet + ne + verbe être + pas + adjectif

Consigne : Transformez ces phrases à la forme négative

1. Je suis content. →

2. Nous sommes fatigués. →

3. Elle est forte. →

4. Ils sont amusants. →
